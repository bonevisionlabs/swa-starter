What's Included

Security Headers

HSTS, CSP, X-Frame-Options, Referrer-Policy, and Permissions-Policy pre-configured. Blocks sensitive file types (.py, .env, .pem) by default.

SSO Authentication

Sign in with Microsoft, GitHub, or Google via Azure SWA's built-in auth. Role-based access control for protected routes.

CI/CD Pipeline

GitHub Actions workflow that deploys on push to main. Path-filtered triggers so unrelated changes don't redeploy. PR preview environments included.

Dark/Light Mode

Theme toggle with localStorage persistence and flash-free loading. CSS custom properties make it trivial to customize your palette.

Route Rules

Clean URLs, SPA fallback, auth redirects (401 → /login), and file blocking rules ready to customize for your app.

Azure Functions API

Optional api/ directory for serverless backend functions. Pre-configured in the workflow and route rules.
